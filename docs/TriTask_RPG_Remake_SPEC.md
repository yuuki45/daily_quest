# TriTask RPGリメイク版 企画・仕様まとめ

## 1. プロジェクト概要

### 旧アプリ名
TriTask

### リメイク版の方向性
TriTaskを、レトロRPG風の世界観を持つToDo・習慣化アプリとしてリメイクする。

ただし、特定の既存ゲーム作品を再現・模倣するのではなく、  
**「毎日のタスクを冒険に変える、オリジナルのレトロRPG風タスク管理アプリ」**として制作する。

---

## 2. コンセプト

### 一言コンセプト
毎日のタスクを、今日の冒険に変えるアプリ。

### コア体験
ユーザーは毎日、最大3つのタスクを「クエスト」として設定する。

- メインクエスト：今日一番やるべき重要タスク
- サイドクエスト：余裕があれば達成したい補助タスク2つ

タスクを完了すると経験値が入り、レベルアップや冒険記録、週ボスの討伐に反映される。

---

## 3. TriTaskから引き継ぐ思想

TriTaskの核は、以下のシンプルな構造にある。

- 1日のメインタスクは1つに絞る
- サブタスクは2つまで
- やることを増やしすぎず、継続しやすくする
- 小さな達成感を毎日積み上げる
- ストリークや達成演出で継続を促す

リメイク版では、この構造をRPG風に変換する。

| 旧TriTask | RPGリメイク版 |
|---|---|
| メインタスク | メインクエスト |
| サブタスク | サイドクエスト |
| タスク完了 | クエスト達成 |
| ストリーク | 連続冒険日数 |
| 継続 | レベルアップ |
| バッジ | 称号・宝箱・装備風アイコン |
| 履歴 | 冒険の記録 |
| 達成演出 | EXP獲得・宝箱・ボスHP減少 |

## 7. MVP方針

最初のリリースでは、ゲーム要素を盛りすぎない。

### MVPで実装する機能
1. 今日のメインクエストを1つ登録できる
2. サイドクエストを2つ登録できる
3. クエストを完了できる
4. 完了時に経験値を獲得する
5. レベルと経験値バーを表示する
6. 連続冒険日数を表示する
7. 今週のボスを表示する
8. メインクエスト達成でボスHPが減る
9. 冒険記録を確認できる
10. レトロRPG風のUI・演出を入れる

### MVPで後回しにする機能

## 8. メイン画面仕様

### 画面名
冒険 / 今日のクエスト

### 目的
ユーザーが今日やるべき3つのクエストを確認し、完了できるメイン画面。

### 表示要素

#### ヘッダー
- タイトル：今日のクエスト
- サブコピー：毎日のタスクを冒険に変える
- 小さな剣・盾・星などの装飾

#### ステータスカード
表示する情報：

- アバター画像
- Lv.12
- EXPバー
- EXP値：70/100
- 連続冒険日数：5日

#### メインクエストカード
- 見出し：メインクエスト
- チェックボックス
- タスク名
- 例：アプリのLPを30分作る

#### サイドクエストカード
- 見出し：サイドクエスト
- チェックボックス付きタスクを2件表示
- 例：
  - 散歩する
  - Xに投稿する

#### 今週のボスカード
- 見出し：今週のボス
- ボス名：先のばしドラゴン
- HPバー：7/10
- オリジナルのかわいいボス画像
- 宝箱アイコンなどの装飾

#### 追加ボタン
- テキスト：クエストを追加
- 大きめのCTAボタン
- 青系またはゴールド系の配色

#### ボトムナビゲーション
4タブ構成。

| タブ | 役割 |
|---|---|
| 冒険 | 今日のクエスト画面 |
| 記録 | 冒険履歴・カレンダー |
| ボス | 週ボス・目標管理 |
| 設定 | テーマ・通知・その他 |

| メインクエスト達成 | 50 |

### 1日最大EXP

## 12. 連続冒険日数仕様

### 理由

### 表示例


## 13. 週ボス仕様

### 目的
1週間単位の継続目標を可視化する。

### 基本仕様
- 毎週1体のボスが出現する
- メインクエストを達成するとボスHPが1減る
- 週7日でHP7のボスを倒すイメージ
- サイドクエスト達成で追加ダメージや宝箱ゲージ加算も可能

### 表示例
```text
今週のボス
先のばしドラゴン
HP 7/10
```

### ボス名案
- 先のばしドラゴン
- めんどくさゴブリン
- 集中力どろぼう
- 三日坊主ゴースト
- 夜ふかしバット
- だらけオーガ

### 注意
敵キャラは完全オリジナルにする。既存ゲームのモンスターに似せない。

## 25. 最重要方針

このアプリで一番大事なのは、RPG要素の多さではない。

一番大事なのは、  
**ユーザーが毎日アプリを開いて「今日も少し進めよう」と思えること。**

そのため、以下を優先する。

1. タスク入力が簡単
2. 完了が気持ちいい
3. 成長が見える
4. 世界観が楽しい
5. 継続のハードルが低い
6. 既存作品に依存しないオリジナル性がある

## 27. まとめ

TriTaskのリメイク版は、  
**「1日3つのタスク」×「レトロRPG風の冒険体験」**を軸にする。

単なるToDoアプリではなく、  
毎日の小さな行動を「クエスト」として意味づけし、  
達成感・成長感・継続の楽しさを提供するアプリを目指す。

MVPでは機能を絞り、まずは以下の体験を完成させる。

> 今日のメインクエストを1つ決める。  
> サイドクエストを2つ添える。  
> 達成すると経験値が入り、少しずつレベルアップする。  
> 週のボスを倒しながら、自分の冒険を続けていく。

これがリメイク版TriTaskの中心体験である。
